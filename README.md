# Gym Supplements Store

A responsive e-commerce product listing website 
built with pure HTML, CSS Grid and Media Queries.

## Features
- Product cards with real images
- Product name, description and pricing
- Buy Now button on each product
- 2 column layout on desktop
- Single column layout on mobile
- Responsive using CSS Media Queries

## Technologies Used
- HTML5
- CSS3
- CSS Grid
- Media Queries

## How to Run
Download all files and open 
response.html in your browser
